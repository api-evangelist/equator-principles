# Equator Principles (equator-principles)
The Equator Principles is a risk management framework adopted by financial institutions for determining, assessing, and managing environmental and social risk in project finance transactions. It provides a minimum standard for due diligence and monitoring to support responsible risk decision-making and is primarily intended to provide a minimum standard for due diligence and monitoring in project finance and related advisory activities.
